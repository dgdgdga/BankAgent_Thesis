\begin{center}
{\fontsize{16}{19}\selectfont\textbf{Appendix A: Database Schema}}
\end{center}

\vspace{3mm}

This appendix documents the complete database schema for the agent\_platform database. All tables use InnoDB engine, utf8mb4 character set with utf8mb4\_unicode\_ci collation. The schema comprises 14 tables across six functional domains.

\vspace{4mm}
